Flexbox

Introduction

Flexbox helps us align and distribute items within a container, along one direction (or main axis) at a time.

Properties for container

To set up a flexbox, we need these properties:

  • display value of flex: behaves like a block-level element itself and inline-flex makes a container behaves like an inline element.
  • flex-flow is a short-hand property for flex-wrap and flex-direction.
    • flex-wrap defines how the flex-item is wrapped or not. It also set the direction of the text which is wrapped.

      Its possible values are:

      • nowrap(default value)
      • wrap
      • wrap-reversed
  • flex-direction defines how the flex item is placed in a flex container defining the main axis and the direction.

    Its possible values are:

    • row
    • row-reverse
    • column
    • column-reverse
  • align-item sets the align-self value on all direct children as a group. In flex box, it controls the alignment of an item on the cross axis.

    Its possible values are:

    • stretch
    • center
    • start
    • end
  • justify-content control how the space between or around content item is along the main axis of a flex container.

    there are five possible value for this property:

    • center
    • start
    • space-between
    • space-around
    • space-evenly
  • align-content(for wrapping flex container) sets the space between or around the flex item along the cross axis.

    It has four possible values:

    • start
    • center
    • space-between
    • space-around

Using flex and it's properties

  • flex controls how a flex item grows or shrinks to fit the flex container. To make flex shrink by its minimum content size, we need to set min-height or min-height.
    • flex-grow controls the growth(width and height) of a flex item.
    • flex-shrink makes a flex item becomes smaller.
    • flex-basis sets the initial main size of a flex item.
  • align-self controls the alignment of inside the grid area. This property doesn't apply to block-level boxes, or to a table cells. It is ignored if flex box margin is auto
  • order property orders flex items inside a flex box or grid.

Terms or concepts

  • Flex container is the container which expand or shrink an item to fill the available space.
  • Flex item is an element which is inside the flex container.
  • Main axis is the vertical axis or y.
  • Cross axis is the horizontally axis which also known as x
  • flex-start is left side in the flex container for the cross axis.
  • flex-end is the right side in the flex container for the cross axis.

The answers of the questions

  1. The order of property values in the shorthand flex property:
    1. flex-grow
    2. flex-shrink
    3. flex-basis
    In the flex: 2 1 auto, 2 presents flex-grow, 1 is flex-shrink and auto presents flex-basis.
  2. We need flex-wrap: wrap or row wrap we can use align-content
  3. Differnce between align-content and align-items:
    • align-content align the content inside the flex box and control the space between items.
    • align-items changes the items height or position inside the flex box.
  4. The flex item's align-self overrides the other.
  5. min-width is similar to flex-basis.
  6. There are two ways to change the ordering of flex items within a flex container. One is done on each flex item by using the order property. The flex-direction:row-reverse or column-reverse can change the whole order of the flex items at the container level.
  7. If we want some space between each of our three flex items in a flex column, we can use flex-wrap: wrap, justify-content: space-between and justify-content: space-around.
