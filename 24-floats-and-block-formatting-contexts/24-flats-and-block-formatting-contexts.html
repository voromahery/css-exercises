Floats, flow and formating contexts

Floats

We need to know the following properties:

  • float
  • clear

  1. their default values are none.

  2. The possible values for clear are:

    • left
    • right
    • both
    • inherit
    • initial

  3. The rule associated with a floated item in terms of source order is, the float item must be before element float over.

  4. Yes, float can float other elements besides images.

  5. Yes, there are a way to create a two-column layout with floats by using the value left and right values.

BFC(Block formating context)

  • BFC is all about layout or a set of rule of how the box interact to another.

  • We can clear a float creating a block formatting context on the parent, this is achieved by putting a certain set of properties on the container. we can use display: flow-root and position: absolute.
