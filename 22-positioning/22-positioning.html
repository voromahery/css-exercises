Positioning

Introduction

Positioning is a layout-breaking form of layout. i.e. it pulls elements out of the normal flow, and one has to use known dimensions to account for these positioned elements.

  • relative
  • absolute
  • fixed
  • sticky

The default value for position is static

The answers of the questions

  1. When using non-default value, we need top, left, right and bottom properties.

  2. The inferred display value of an element that has been positioned is block.

  3. relative still maintains a box in the flow of the document.

  4. The special or significant about fixed and sticky is, they always appear even if the user scroll the page and their positions don't change.

  5. When positioning an element absolutely, we can lock that element's coordinates to its parent element by positioning the parent element into relative.

  6. absolute is useful for positioning its nearest ancestor. It also helps us to position an element precisely where we want to put it.

  7. Sometimes if you put two absolutely positioned elements in the same physical space, they'll stack, i.e. only one will be visible on top.

  8. CSS Stacking Context is an element that contains a set of layers or the element jusxtaposed by using z-index property.
