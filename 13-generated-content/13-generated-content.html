Counters

To count on CSS, we use counters. To use counters, we have to work with these properties:

  • counter-reset (Creates or resets a counter)
  • counter-increment (Increments a counter value)
  • content (Inserts generated content)
  • counter( ) or counters( ) function (Adds the value of a counter to an element)

We reset a counter by using counter-reset property.

Counters are useful for counting the number of elements and to refer each division in a document. It also helps us to style a list.

Lists

The properties that only related to list in CSS are:

  • list-style-type
  • list-style-position
  • list-style-image

The type of bullet style are:

  • Disc
  • Image
  • Circle
  • Square
  • Upper-roman
  • Lower-alpha
  • Decimal
  • Decimal-leading-zero
  • Lower-roman
  • Upper-roman
  • Lower-greek
  • Lower-latin
  • Upper-latin
  • Armenian
  • Georgian
  • None

Any group of elements can be styled to look like a list by using counters.
