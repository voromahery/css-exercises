The Calc function

  1. calc() function supports number, second, minute, frequency, fraction, keywords value.
  2. To control the mathematical order of operation (PEDMAS / BODMAS) in the function, so that something like calc(2 + 3 * 4) doesn't equal 14, when it is instead meant to equal 20 is, by putting the 3 * 4 inside brackets like calc((2 + 3) * 4).
  3. We can use a calc() function to convert one type of unit to another type of unit.
  4. If we want to convert a unitless number to a unit value, we have to multiply or divide that number with 1 with a unit.
  5. The plus and minus operators inside the function need spaces around them because if not, the calc() function considers it as a positive value for the plus and negative for the minus.
  6. We need to multiply or divide a value into -1 if we want to convert a value into a negative value.
  7. calc() function works with colour values such as a light red plus a light red should equal a darker red.
