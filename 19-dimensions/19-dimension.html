Dimensions

Properties:

  • width: auto
  • height: auto
  • min-width: 0
  • max-width: none
  • min-height: 0
  • max-height: none

These values doesn't change if the element is a block-like or an inline-like element because inline-like or block-like elements maintain their width and height.

If the min-width is greater than width, the min-width controls the size of an element.

If the max-width is less than a width, the size of an element is set by the width.

If the min-width is greater than max-width, the size of an element is controled by the min-width.

Percentage doesn't work with a height, but it can work with width.

div { min-width: 20rem; max-width: 80vw; }

In this ruleset, it seems that the unit vw is less than rem, so the value rem is applied.

The implied width of a div is min-width: 20rem.
