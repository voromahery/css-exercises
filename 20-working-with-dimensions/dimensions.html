Images should fit

Set a max-width on img elements so that they fit within the viewport width.

Make sure the height is correct too.

Avoid unnecessary wrapping

Sometimes we'll want to preserve whitespace, and the number of linebreaks in something like a table.

In other words, we don't want the text to wrap too much. Use a min-width property to control this in a table.
