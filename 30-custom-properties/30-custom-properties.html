Custom Properties

  1. When defining a Custom Property's name, we have to use -- or double dashes as a prefix.
  2. We can't use *, #, /, \, ( and ).
  3. We can use numbers or a number with unit, keywords such as color annotations and variation of fonts to assign to a new Custom Property.

  4. It is useful to create Custom Properties using the :root or html if we want to style globally all HTML elements within a document.

    It isn't useful when we want to style specifically each element in a document.

  5. After we've defined a new Custom Property, we can use it as a value of another properties in a function.

    The function that we use is var(custom property name).

  6. The ways that override or change a Custom Property are:
    • importance by using a keyword important!
    • Specificity of the selector by using ID
    • Source Order of an element
  7. Custom Properties are useful because they help us to write less code and to avoid repetition of a code.
  8. The only aspect (or part) of CSS syntax that we can use Custom Properties for is the function calc().
