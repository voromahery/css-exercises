Inheritance

What is CSS inheritance?

When other element inherits by default the value of properties from another element, that is called inheritance. It affects the child element of a parent element which has a value of a property.

The property which have values that are inherit by default are:

  • border-spacing
  • caption-side

CurrentColor and inherit

The currentcolor is expected to change or done in purpose. However, inherit is by default and have the same value as its parent element.

What do 'revert', 'initial' and 'unset' keyword do?

  • Revert makes one element to go back to its inherited value value.
  • Initial makes an element to swich back to its default value.
  • Unset resets a property to its default value.

Some properties are not applied to them so, it may be dangerous to use them.

The possibilities with using an ancestor universal selector combined with the inherit are, you can change only the selected element to be the same as its parent element or the same as its root element.
