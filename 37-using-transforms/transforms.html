Using translate() to show or hide

We can use translateX to hide a menu away behind the user-interface, when the checkbox is unchecked.

But when the user clicks a label that toggles the checkbox, we can use that new checkbox state to act as the open signifier for the menu.

When that happens, we reset the nav's translateX.

3D transforms

Let's build a 3D dice (a cube).

The cube's size should be var(--face-size) * var(--face-size) * var(--face-size).

  1. Start by placing the 1 and 6 on either sides of a 3D cube by using translateZ().
  2. You can click the checkbox to 'spin' the cube. This will allow you to see how your transforms are working along the way.
  3. Then try rotating the 2 and 5 along the correct axis (X, Y or Z?), and translating them into the correct position, either before or after the rotate.
  4. Lastly, use a similar technique for the 3 and 4, but over different axes.
