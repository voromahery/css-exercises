Box model

Box model is used to create an image which is shaped like a box or table in CSS. Box model has a few properties that are needed to create a box. Those properties are margin which cover the border and is often transparent, the border which contains the padding and the padding which contains the content of a document.

Diagram of the box model

Content box

Content box is all element which has resemblance on box, it also has contents in it.

Padding box

Padding box is the clear space inside the border.

Border box

Border box is the visual limit of the box, it can be transparent but most of the time it is visible. Border is wrapped by margin.
