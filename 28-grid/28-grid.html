Grid layout

Basic properties

  • display: grid makes the grid items into rows with their full width by default.
  • display: inline-grid present the grid into column and the width of the rows are controled by the width of the contents.
  • grid it is a shorthand property that sets grid-template-columns, grid-template-rows, grid-template-areas, grid-auto-rows, grid auto-columns and grid-auto-flows.
    • grid-template-columns defines the size of items in a column.
    • grid-template-rows resizes items in a row
    • grid-gap sets the space between each rows or columns.
      • grid-row-gap sets gap between each rows.
      • grid-column-gap sets gap between two or more columns.
  • grid-auto-columns makes a grid wider.
  • grid-auto-rows resizes the height of a grid.

Other properties apply to the grid

Some of these properties tell a grid item where to start and end in terms of columns and rows.

  • grid-column is a shorthand property for grid-column-start and grid-column-end.
    • grid-column-start specifies where to start in the grid column.
    • grid-column-end specifies the end position of a grid item in a grid column.
  • grid-row is a shorthand property for grid-row-start and grid-row-end.
    • grid-row-start specifies where to start within the grid row.
    • grid-row-end specifies the end position of a grid item in a grid row.

We can also start naming certain areas in the whole grid

grid-template-areas

...and then assign a grid-item to a named area:

grid-area

Lastly, we can change the algorithm CSS grid uses to fill the available space using grid-auto-flow.

Keywords

  • repeat() used in grid rows or columns to present a repeated fragment.
  • minmax() defines the size of a grid equal to max or less and equal to min or greater.

The answers of the questions

  1. By default, a grid has one column
  2. The difference between Flexbox and Grid:
    • Flexbox is used to design a layout in one dimension. It means, row or column
    • Grid used to desing both rows or columns at the same time.
  3. I would use flexbox rather than grid when I want to design a single dimension layout.
  4. I would use grid rather than flexbox when I want to design a table-like layout.
  5. A more concise way to write out the following: grid-template-columns: 10rem 10rem 10rem 10rem; is: grid-template-columns: repeat(4, 10rem)
  6. We can use grid-template-rows to specify a minimum height for a grid row, but still allow the row to increase in height if its content is too tall.
